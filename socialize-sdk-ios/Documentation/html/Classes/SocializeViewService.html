createViewForEntity:longitude:latitude:

This method creates view for entity.

- (void)createViewForEntity:(id<SocializeEntity>)entity longitude:(NSNumber *)lng latitude:(NSNumber *)lat

Parameters

entity

SocializeEntity object which should be marked as viewed.

lng

Longitude float value. Could be nil. (OPTIONAL)

lat

Latitude float value. Could be nil. (OPTIONAL)

Discussion

Successful call of this method invokes [SocializeServiceDelegate service:didCreate:] method. In case of error it will be called [SocializeServiceDelegate service:didFail:] method.
