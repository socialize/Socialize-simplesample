authenticateWithApiKey:apiSecret:thirdPartyAppId:thirdPartyLocalAppId:thirdPartyName:

Third party authentication via SDK service.

- (void)authenticateWithApiKey:(NSString *)apiKey apiSecret:(NSString *)apiSecret thirdPartyAppId:(NSString *)thirdPartyAppId thirdPartyLocalAppId:(NSString *)thirdPartyLocalAppId thirdPartyName:(SocializeThirdPartyAuthType)thirdPartyName

Parameters

apiKey

API access key.

apiSecret

API access secret.

thirdPartyAppId

Extern service application id.

thirdPartyLocalAppId

Extern service local application id.

thirdPartyName

Third party authentication name. Current verion of SDK suports only FacebookAuth value.

Discussion

Third Party Authentication uses a service like Facebook to verify the user. Using third party authentication allows the user to maintain a profile that is linked to their activity. Without using Third Party Authentication, the user will still be able to access socialize features but these actions will not be linked to the user’s profile.

Find information how to get API key and secret on Socialize.

Note: In current SDK version only Facebook authentication is available.
